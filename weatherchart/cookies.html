Clear control

Cookie and privacy choices.

WeatherChart does not use analytics or advertising cookies. It uses cookie-free Cloudflare Web Analytics for aggregate page-view and performance measurement. One necessary browser-storage record remembers whether you allow the optional external interactive map.

Your current control

Allow, reject or revisit the map choice

The optional map is off until you make a choice. Rejecting it leaves forecasts, warnings, charts and accessible tables available. Allowing it lets this browser request the Leaflet map library and OpenStreetMap tiles.

Your saved choice, if any, can be changed at any time.

At a glance

  • Cookie-free Cloudflare analytics beacon

External services

What connects, and when

Cloudflare Web Analytics loads on each page to measure aggregate visits and page performance without analytics cookies or browser storage.

If you allow the map, the browser downloads Leaflet 1.9.4 from unpkg and requests map tiles from OpenStreetMap. Those providers receive ordinary connection information such as your IP address and browser headers under their own policies. If you reject or later withdraw permission, WeatherChart removes the map and stops further tile requests on that page.

Other connections happen only when needed for a feature you choose: an uncached place search contacts Open-Meteo geocoding or postcodes.io, and selecting an attributed external link opens that provider. The site also reads its own generated weather JSON from WeatherChart UK.
